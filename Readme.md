# Hobi amaçlı geliştirdiğim isometric 2D mmorpg oyunu
## Geliştirme amacı.
Bu oyun şahsım tarafından sadece hobi amaçlı geliştirilmekte olup hiçbir kãr amacı gütmeyip dümdüz eğlence odaklıdır.
İsteyen herkes kendi serverını kurup bu eğlenceye dahil olabilir. Oyun için düşündüğüm teknoloji ağacı aşşağıdaki gibidir.
## Teknoloji Ağacı
### Backend:
Golang
Fiber
Gorm
Sqlite
Testify
#### Game Engine Backend Side
